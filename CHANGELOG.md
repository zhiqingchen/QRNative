# Changelog

All notable changes to QRNative will be documented here.

## Unreleased

- Ship a universal binary so the app runs on both Apple Silicon and Intel Macs.
- Initial native macOS SwiftUI app.
- Added QR generation from typed text and clipboard text.
- Added searchable local history stored in Application Support.
- Added QR recognition from image files, dropped images, and clipboard images.
- Added selected-text QR generation, selected-image recognition Services, and screenshot QR recognition.
- Added configurable global shortcuts and native Settings panes.
- Added app bundle and DMG packaging scripts.
- Added PNG export, image copy, decoded text copy, and URL opening.
- Added app icon, project branding, and open-source repository metadata.
