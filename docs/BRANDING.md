# QRNative Branding

## Name

Use `QRNative` as the product and repository name.

## Logo

- Source SVG: `Assets/Brand/qrnative-logo.svg`
- App icon: `Resources/QRNative.icns`
- Style: minimal abstract macOS utility mark, rounded square, three QR-inspired finder nodes, one quiet scan line.

## Colors

- Ink: `#0B1020`
- Deep Blue: `#1A2D52`
- Paper: `#F8FAFC`
- Scan Mint: `#30E7C2`
- Soft Periwinkle: `#8EA2FF`

## Usage

- Use the generated PNG in the README so GitHub renders it consistently.
- Use the SVG as the editable source for brand and web contexts that need vector output.
- Prefer the `.icns` file for app bundles.
- Keep clear space around the mark equal to at least 12% of the logo width.
- Do not place the logo on busy backgrounds.
- Keep the mark minimal; avoid decorative stars, full QR grids, and extra module dots in brand contexts.

## Short Description

Native macOS QR code generator, history manager, and recognizer.

## Long Description

QRNative is a privacy-friendly native macOS utility for generating QR codes from text or clipboard content, keeping searchable local history, exporting crisp PNGs, and recognizing QR codes from images.
